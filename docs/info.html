  • in src/routes/custom-event/+server.js

    export function POST() {

    return produce(async function start({ emit }) {

  • In page.svelte:

import { source } from 'sveltekit-sse'

 const value = source('/custom-event').select('message')

  1. Websocket

Siehe suhaildawood/SvelteKit-integrated-WebSocket: First-class support for WebSockets within SvelteKit by attaching a WebSocket server to the global state.



  • Da unser Projekt als javascript angelegt wurde, müssen Datei-Endung und manche Stellen angepasst werden:

  • entferne aus scripten import type...

  • Variablen-Typangaben und ...as xxx

  • npm install ws



  • package.json anpassen:

    • in devDependencies hinzufügen "@types/ws": "^8.5.5",

    • ins scripts hinzufügen (das ist der Server wenn ein Produktionsbuild gemacht wird): "prodServer": "PORT=3005 tsm ./prodServer.ts"

  • geänderte Abhängigkeiten installieren mit npm install

  • prodServer.js anlegen (direkt im Hauptverzeichnis)
